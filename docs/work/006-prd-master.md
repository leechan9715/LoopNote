## 1. 서비스 개요
루프노트는 학생이 모르는 문제를 사진으로 찍으면, AI가 실시간으로 분석하여 10분 내에 스스로 풀 수 있도록 단계별 힌트와 미션을 제공하는 '오답 회복 플랫폼'입니다.

---

## 2. 사용자 기능 명세 (Feature List)

### 2.1 학생용 (Next.js Web App)
*   **[F-01] 오답 스캔**: 카메라를 이용해 문제지 사진 촬영 및 텍스트/수식 인식 (OCR).
*   **[F-02] AI 미션 생성**: 인식된 문제의 개념을 파악하여 10분 분량의 단계별 힌트 미션 생성.
*   **[F-03] 인터랙티브 학습**: 학생이 입력한 중간 풀이 과정을 AI가 실시간 피드백.
*   **[F-04] 보상 시스템**: 미션 완료 시 '회복 에너지' 포인트 및 캐릭터 배지 지급.
*   **[F-05] 나의 회복 노트**: 완료한 미션들을 개념별로 모아보는 아카이빙 기능.

### 2.2 학부모용 (Next.js Dashboard)
*   **[F-06] 실시간 알림**: 자녀가 미션을 시작하거나 완료했을 때 푸시 알림.
*   **[F-07] 주간 회복 리포트**: 자녀의 취약 개념, 학습 시간, 성취도 시각화 대시보드.
*   **[F-08] 프리미엄 결제**: 정기 구독 서비스 결제 및 멤버십 관리.

---

## 3. 비즈니스 로직 및 정책
*   **미션 제한**: 무료 사용자는 일 3회 미션 가능 (자정 기준 초기화).
*   **AI 힌트 정책**: 직접적인 정답 노출은 3단계 힌트 이후에만 가능하도록 설계.
*   **데이터 보존**: 개인정보 보호를 위해 학생의 풀이 사진은 3개월 후 자동 익명화 처리.

---

## 4. 기술 제약 사항
*   **OCR 성능**: 필기체 인식률 90% 이상 확보 필요 (Google Vision API 등 검토).
*   **AI 응답 속도**: 힌트 생성 및 피드백 대기 시간 3초 이내 목표.
*   **반응형 웹**: 모바일 기기(태블릿 포함) 환경 최적화 필수.
